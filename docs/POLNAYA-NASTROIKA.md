## ЧАСТЬ 2. Cloudflare (хостинг сайта)

### 2.1. Войти в Cloudflare

1. Открой: `https://dash.cloudflare.com`
2. Войди в аккаунт (email, который привязан к Cloudflare).
3. На главной странице дашборда ты увидишь список сайтов/проектов.

### 2.2. Открыть проект rainclien

1. В левом меню найди раздел **«Build»** (Сборка).
2. Нажми на **«Workers & Pages»**.
3. В списке проектов найди **«rainclien»** (или название с `rainclien.pages.dev`).
4. Нажми на название проекта **rainclien** левой кнопкой мыши.
5. Откроется страница проекта. Вверху должно быть: **Production** и домен **rainclien.pages.dev**.

### 2.3. Проверить подключение к GitHub

1. На странице проекта **rainclien** нажми вкладку **«Settings»** (Настройки) вверху.
2. В левом подменю нажми **«Build»** (если есть).
3. Найди блок **«Git repository»**.
4. Должно быть написано: `sergeew0606-lab/rainclien`.
5. **Production branch** должен быть: `main`.
6. **Build command:** `npm run build`
7. **Build output directory:** `dist`

Если репозиторий не подключён:
1. Нажми **«Connect to Git»** / **«Connect Git repository»**.
2. Выбери GitHub → репозиторий **rainclien** → ветка **main** → Save.

### 2.4. Добавить три секрета (переменные окружения)

1. На странице проекта **rainclien** нажми вкладку **«Settings»** вверху.
2. В левом подменю прокрути и нажми **«Environment variables»**  
   ИЛИ на странице Settings прокрути вниз до блока **«Variables and Secrets»**.
3. Убедись, что сверху в выпадающем списке выбрано **«Production»** (Продакшн), не Preview.

#### Секрет №1 — ЮMoney

1. В блоке **«Variables and Secrets»** нажми синюю кнопку **«+ Add»** (Добавить).
2. Откроется панель справа **«Variables and Secrets»**.
3. **Type:** выбери **Secret**.
4. **Variable name:** введи точно (копируй):  
   `YOOMONEY_NOTIFICATION_SECRET`
5. **Value:** вставь секрет из ЮMoney (тот, что давал ЮMoney при настройке HTTP-уведомлений).
6. Нажми **«Add variable»** или **«Save»**.

#### Секрет №2 — адрес базы Firebase

1. Снова нажми **«+ Add»**.
2. **Type:** можно **Secret** или **Text**.
3. **Variable name:**  
   `FIREBASE_DB_URL`
4. **Value:**  
   `https://rainclient-default-rtdb.firebaseio.com`
5. Сохрани (**Add variable** / **Save**).

#### Секрет №3 — пароль базы Firebase

1. Снова нажми **«+ Add»**.
2. **Type:** **Secret**.
3. **Variable name:**  
   `FIREBASE_DB_SECRET`
4. **Value:** секрет из Firebase (см. Часть 3.3 ниже).  
   Пример формата: длинная строка букв и цифр из **Database secrets**.
5. Сохрани.

### 2.5. Сохранить все изменения

1. Если справа открыта панель с переменными — нажми синюю кнопку **«Save»** внизу справа.
2. Дождись сообщения об успешном сохранении.

### 2.6. Пересобрать сайт (обязательно после секретов)

1. Вверху страницы проекта нажми вкладку **«Deployments»** (Развёртывания).
2. Найди самый верхний деплой с меткой **«Production»**.
3. Справа от него нажми на три точки **「⋯」**.
4. В меню выбери **«Retry deployment»** (Повторить развёртывание).
5. Подожди 1–3 минуты, пока статус не станет зелёным (Success / Успешно).
6. Сайт будет доступен по адресу: **https://rainclien.pages.dev/**

### 2.7. Постоянная ссылка (чтобы не менялась)

1. Всегда используй только: **https://rainclien.pages.dev/**
2. **Не используй** ссылки вида `https://e750ab73.rainclien.pages.dev` — это preview, они меняются каждый раз.
3. В ЮMoney и для покупателей давай только **https://rainclien.pages.dev/**

## ЧАСТЬ 3. Firebase (база данных)

### 3.1. Войти в Firebase

1. Открой: `https://console.firebase.google.com`
2. Войди в Google-аккаунт, к которому привязан проект.
3. На главной странице Firebase Console увидишь список проектов.

### 3.2. Открыть проект rainclient

1. Найди плитку/карточку проекта **«rainclient»**.
2. Нажми на неё.
3. Откроется обзор проекта (Project Overview).

### 3.3. Скопировать Database secret (для Cloudflare)

1. Слева вверху нажми на **иконку шестерёнки** рядом с надписью **«Project Overview»**.
2. В выпадающем меню выбери **«Project settings»** (Настройки проекта).
3. Откроется страница настроек. Вверху вкладки: General, **Service accounts**, и др.
4. Нажми вкладку **«Service accounts»**.
5. В левой колонке этой страницы найди **«Database secrets»** (Секреты базы данных) под разделом Legacy.
6. Или открой прямую ссылку:  
   `https://console.firebase.google.com/project/rainclient/settings/serviceaccounts/databasesecrets`
7. В таблице напротив **rainclient-default-rtdb** будет строка **Secrets**.
8. Нажми на иконку **глаза** (показать) или **копировать**.
9. Скопируй длинную строку — это значение для **FIREBASE_DB_SECRET** в Cloudflare (Часть 2.4, секрет №3).

### 3.4. Вставить правила безопасности (Rules)

1. В левом меню Firebase нажми **«Build»** (Создать) или раздел с базами.
2. Нажми **«Realtime Database»**.
3. Если база ещё не создана — нажми **«Create Database»**, выбери регион, режим (можно test, потом сменишь rules).
4. Вверху страницы базы нажми вкладку **«Rules»** (Правила).
5. Удали всё, что там написано (особенно старые строки `now < 1781557200000`).
6. Вставь целиком этот текст:

```json
{
  "rules": {
    "users": {
      "$username": {
        ".read": true,
        ".write": true
      }
    },
    "keys": {
      "$key": {
        ".read": true,
        ".write": true
      }
    },
    "promos": {
      "$code": {
        ".read": true,
        ".write": false
      }
    },
    "payments": {
      "$id": {
        ".read": true,
        ".write": "!data.exists() && newData.child('status').val() === 'pending'"
      }
    }
  }
}
```

7. Нажми кнопку **«Publish»** (Опубликовать) вверху справа.
8. Дождись сообщения, что правила опубликованы.

### 3.5. Создать промокод (необязательно)

1. В Realtime Database нажми вкладку **«Data»** (Данные).
2. Если нет узла **promos** — нажми **+** у корня базы, имя: `promos`.
3. Наведи на **promos**, нажми **+**, имя кода: например `SALE50`.
4. Значение: `50` (скидка 50%) или объект `{ "discount": 50, "label": "50% OFF" }`.

### 3.6. Проверить данные после оплаты

1. **Data** → папка **payments** → после покупки появится `RC-PAY-...`.
2. Сначала `status: "pending"`, после webhook — `status: "paid"`.
3. В **users** → твой ник → `subscription` с активной подпиской.

## ЧАСТЬ 4. ЮMoney (приём денег и подтверждение оплаты)

### 4.1. Войти в кошелёк

1. Открой: `https://yoomoney.ru`
2. Войди в кошелёк **4100 1195 4107 0621** (тот, что в коде сайта).

### 4.2. Открыть настройки HTTP-уведомлений

**Способ А — прямая ссылка (лучше с компьютера):**

1. Открой: `https://yoomoney.ru/transfer/myservices/http-notification`

**Способ Б — через меню:**

1. Нажми на **аватар / имя** в правом верхнем углу.
2. Выбери **«Настройки»** или **«Настройки кошелька»**.
3. Прокрути список, нажми **«Больше»**, если есть.
4. Ищи пункт **«HTTP-уведомления»** или **«Уведомления о входящих переводах»**.

**Важно:** раздел **«Уведомления»** (письма и SMS) — это **не то**. Нужны именно **HTTP-уведомления**.

### 4.3. Заполнить HTTP-уведомления

1. **Включить уведомления:** галочка / переключатель **Вкл**.
2. **URL для отправки уведомлений** (адрес сайта для webhook), вставь точно:  
   `https://rainclien.pages.dev/api/yoomoney-webhook`
3. **Секрет для проверки подлинности** — ЮMoney покажет или попросит создать.  
   Скопируй его → тот же вставь в Cloudflare как **YOOMONEY_NOTIFICATION_SECRET** (Часть 2.4, секрет №1).
4. Нажми **«Сохранить»** / **«Готово»**.

## ЧАСТЬ 5. Как работает оплата на сайте

1. Пользователь заходит на **https://rainclien.pages.dev/**
2. **Входит** в аккаунт (регистрация/логин).
3. Нажимает **«Купить подписку»**, выбирает тариф, **«Купить»**.
4. Сайт создаёт запись в Firebase `payments/RC-PAY-...` со статусом `pending`.
5. Открывается **новая вкладка** ЮMoney — пользователь платит.
6. ЮMoney отправляет уведомление на  
   `https://rainclien.pages.dev/api/yoomoney-webhook`
7. Webhook проверяет подпись и сумму, ставит в Firebase `status: "paid"`.
8. Сайт видит `paid` и выдаёт подписку:
   - **Навсегда (lifetime):** до **31.01.2038**
   - **Месяц:** создаётся ключ на 30 дней в `keys/`, сразу активируется
   - **Год:** ключ на 365 дней, сразу активируется
9. Подписка записывается в **users/ник/subscription** в Firebase и в профиле на сайте.

## ЧАСТЬ 6. Тестовая покупка (проверка всего)

1. Открой **https://rainclien.pages.dev/**
2. Зарегистрируйся или войди.
3. Открой покупку подписки, выбери самый дешёвый тариф для теста.
4. Нажми **«Купить»**.
5. В новой вкладке оплати в ЮMoney (можно минимальную сумму, если тариф позволяет).
6. **Не закрывай** первую вкладку сайта — подожди 1–3 минуты, пока идёт «Ожидаем оплату…».
7. Открой Firebase → **Realtime Database** → **Data**:
   - есть `payments/RC-PAY-...`?
   - стало `status: "paid"`?
8. На сайте в **профиле** — подписка активна.
